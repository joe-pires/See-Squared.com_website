# Privacy Policy — Per-Site JavaScript Toggle

Last updated: 2026-05-29

This extension does not collect, store, transmit, or share any personally identifiable information, browsing history, page contents, or any other user data with the developer or any third party.

All settings made through the extension (whether JavaScript is allowed or blocked on a given site) are stored locally by Google Chrome using its built-in content-settings system. These settings remain in the user's own browser profile and are managed by Chrome, not by the extension.

The extension maintains a small local diagnostic error log (up to 50 entries) using `chrome.storage.local` solely to help users report problems. This log never leaves the user's browser unless the user manually copies it via the "Report a problem" button and chooses to send it to the developer by email.

No analytics, telemetry, tracking, advertising, or remote code execution is used. The extension makes no network requests.
